About this template

The chat bubble in the bottom-right corner is a reusable "ask this site" assistant. It answers questions only from this site's own content; the knowledge base is compiled from a folder of markdown and YAML at build time, then handed to a serverless function that grounds every answer in it.

How it works

Content in content/ is compiled into the knowledge base by scripts/build_kb.py. The function at /api/ask calls Claude or OpenAI (auto-detected from the configured API key) and is instructed to answer only from that content, declining anything off-topic. Fork it, drop in your own markdown, and deploy.
