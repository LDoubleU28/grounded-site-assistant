# FAQ

## What does Sam work on?

Sam works on backend services, developer tooling, and small web applications. The current focus is serverless functions and retrieval-grounded assistants.

## What technologies does Sam use?

Sam works mostly with JavaScript and Python, plus SQL for data work. Sam prefers minimal dependencies and tools that are easy to reason about.

## Is Sam available for new work?

## How was this assistant built?

This assistant is a grounded chatbot. It answers only from the content in this site's knowledge base and declines questions outside that scope. It runs on a serverless function that calls a large language model API.
