# About Sam Rivera

Sam Rivera is a software engineer who builds web applications and developer tools. Sam has worked across the stack for about a decade, with a focus on backend services, APIs, and the kind of glue code that keeps systems reliable in production.

Sam likes problems that sit at the boundary between systems: data pipelines that feed user-facing features, build tooling that shortens the feedback loop, and small interfaces that hide a lot of complexity. Recent interests include serverless functions, retrieval-grounded assistants, and keeping dependencies minimal.

Outside of work, Sam writes about engineering tradeoffs, contributes to a few open-source projects, and mentors early-career engineers. Sam is based in a fictional city and is generally reachable through the contact form on this site.
