# Projects

## Acme Docs

A documentation site generator that turns a folder of markdown into a fast static site. Sam built the incremental build pipeline and the search index. The generator emphasizes small output size and zero client-side framework dependencies.

## Tidepool

A lightweight job queue for background work in small services. It uses a single database table as the queue, supports retries with backoff, and ships as a single file with no external broker. Sam maintains it as an open-source project.

## Fieldnote

A note-taking tool for engineers that links notes to code references. Sam prototyped the editor and the linking model. It is an experiment in keeping documentation close to the code it describes.
